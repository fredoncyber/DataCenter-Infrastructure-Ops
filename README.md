# DataCenter-Infrastructure-Ops (Hands‑On Lab Project)

A practical, lab‑style portfolio for **Data Center Technician** skills aligned with Google's role.
Each lab is self‑contained with clear objectives, SOPs, commands, expected outputs, and validation steps.

## Labs Overview
- **01_server_rack_assembly** – Rack assembly & cable management checklist.
- **02_linux_diagnostics** – `server_health_check.sh` and `linux_log_analysis.sh` for OS/hardware troubleshooting.
- **03_network_ops** – Network baseline capture and troubleshooting guide.
- **04_data_security** – Secure media handling, sanitization, and asset decommissioning.
- **05_gcp_automation** – GCP VM status monitor (Python) and backup schedule (cron).

## Quick Start
```bash
git clone https://github.com/fredoncyber/DataCenter-Infrastructure-Ops.git
cd DataCenter-Infrastructure-Ops
chmod +x labs/02_linux_diagnostics/*.sh labs/03_network_ops/net_baseline_capture.sh
```
