# Server Rack Assembly Checklist

- [ ] Confirm rack layout (U positions, airflow front‑to‑back).
- [ ] Install rails and verify alignment.
- [ ] Mount server; tighten to torque spec.
- [ ] Connect power (A/B) to PDUs; label both ends.
- [ ] Connect network cables; verify color/key per VLAN/role.
- [ ] Apply cable strain relief and velcro ties (no zip ties on fiber).
- [ ] Record asset tag, serial, and port map in CMDB.
